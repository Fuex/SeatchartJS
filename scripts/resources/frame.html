Create beautiful designed seat maps, legends and shopping carts.

Usage

First of all link the library along with the stylesheet and the script that generates the seatchart.

            
<link rel="stylesheet" href="/path/to/seatchart.css">
<script type="text/javascript" src="/path/to/seatchart.js"></script>

<script>
    var map = {
        rows: 10,
        cols: 7,
        reserved: [9, 14, 17],
        disabled: [12, 21, 23]
    };

    var types = [
        { type: "regular", color: "orange", price: 10 },
        { type: "reduced", color: "#af0000", price: 7.5 },
        { type: "military", color: "red", price: 7 }
    ];

    var sc = new SeatchartJS(map, types);
    sc.setAssetsSrc("/path/to/assets");

    // Create Functions
    sc.createMap("map-container");
    sc.createLegend("legend-container");
    sc.createShoppingCart("shoppingCart-container");
</script>

Then in you web page body create three containers that are going to contain the three elements: seat map, legend and shopping cart. Their id needs to be the same passed to the create functions, in your script.

            
<div id="map-container"></div>
<div id="legend-container"></div>
<div id="shoppingCart-container"></div>

Enjoy the result.

HOW TO:

  • Right click to select a seat
  • Left click to unselect a seat
  • Long press to change seat type
